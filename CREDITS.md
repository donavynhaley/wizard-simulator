# Credits

## Runtime Third-Party Assets

The curated runtime files are documented in `assets/third_party/README.md`.

### Kenney Fantasy Town Kit

The project uses selected models from the [Kenney Fantasy Town Kit](https://kenney.nl/assets/fantasy-town-kit).
These assets are released under Creative Commons Zero, and the original license is preserved at `assets/third_party/kenney/LICENSE.txt`.

### Poly Pizza Models

The runtime wizard model is stored at `assets/third_party/polypizza/Wizardus Maximus.glb`.
The original local Poly Pizza folder, including the unused `fps_arms.glb`, is retained at `source_assets/third_party_packs/polypizza/` for provenance.

`fps_arms.glb` is Rigged Fps Arms by Player11132 from [poly.pizza](https://poly.pizza/m/AMGNKfQqVc), licensed under CC BY.
It is retained as source material and is not referenced by the current runtime.

### Animated Seal

The runtime copy is stored at `assets/third_party/seal/seal_animated_low_poly.glb`.
The original local file is retained under `source_assets/third_party_packs/`.

## Complete Asset Packs

Complete downloaded packs and their original manifests and licenses are retained under `source_assets/third_party_packs/`.
Godot ignores that directory so unused pack variants do not affect editor imports.
